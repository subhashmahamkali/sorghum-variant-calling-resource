# Profiling Index

This folder provides profiling-oriented indices of code and runtime metadata for
review, reproducibility, and manuscript preparation.

## Files

- `code_inventory.tsv`
  - maps step directories to script files, manuscript wrappers, and workflow RMDs
- `manifest_inventory.tsv`
  - maps key metadata/path manifest files used in execution and reporting

## Intended Use

- Quickly audit which script drives which step.
- Trace sample/path manifests used during HCC execution.
- Support methods/supplementary preparation with explicit file references.
