# Data Index

This folder is a publication-facing index for data products and supplementary
materials.

## Policy

- Keep large primary/intermediate files on HCC (`/work`, `/mnt`).
- Keep only lightweight metadata, manifests, and references in Git.

## Current Linked Sources

- Curated sample metadata: `../metadata/curated/`
- Path manifests: `../metadata/locations/`
- Run snapshots: `../metadata/run_tracking/`

## Supplementary

Use `data/supplementary/` for manuscript-ready supplementary tables/indices.
